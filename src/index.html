Simple LinkedIn Composer

Instructions

Getting Started

  1. Enter your LinkedIn user id.
  2. Compose your snippet. Optional: Press Save Snippet if you want to persist it locally. If you want to use this as a template going forward, check "Mark as template" prior to saving.
  3. Press Copy to Clipboard to copy the post content.
  4. Press Create a new LinkedIn Post to open the LinkedIn post page.
  5. Paste into the LinkedIn textbox, check for accuracy, then post it.

No data leaves your browser. Everything is stored in your browser's local storage.

Background

LinkedIn's post composer doesn't natively allow for rich text formatting, so this tool provides a simple way to compose posts with rich text and emojis. Tools that address custom formatting on LinkedIn are either not free or, if free, not feature-rich. This tool is completely free and open source.

No data leaves your browser. Everything is stored in your browser's local storage.
